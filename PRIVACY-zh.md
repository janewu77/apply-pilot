# 隐私政策 — Apply Pilot

**最后更新：2026 年 4 月 16 日**

Apply Pilot（「本扩展」）是一款 Chrome 扩展，可依据你在本机自行创建并管理的个人档案，自动填写求职申请表单。

---

## 1. 我们收集与存储的数据

Apply Pilot 使用 Chrome 内置的 `chrome.storage.local` API，将以下数据**仅保存在你的设备本地**。除非你主动启用下文第 3 节所述的可选 AI 功能，否则这些数据不会离开你的设备。

| 类别 | 示例 |
|---|---|
| 个人信息 | 姓名、电子邮箱、电话号码、出生日期、性别、国籍 |
| 地址 | 街道、城市、州/省、邮政编码、国家 |
| 在线资料 | LinkedIn、GitHub、作品集、个人网站等 URL |
| 工作信息 | 职位、公司、工作年限、通知期（notice period）、薪资期望、工作许可状态 |
| 教育背景 | 学位、院校、专业、毕业年份 |
| 技能 | 编程语言、技术技能、证书 |
| 预写答案 | 你对常见申请问题预先准备的回答（例如「你为什么想要这个职位？」） |
| AI 服务商设置 | 你选择的 AI 服务商（Anthropic 或 OpenAI）、模型名称、API 密钥 — 仅保存在本地 |

**我们不运营任何服务器。上述数据均不会传送给 Apply Pilot 或其开发者。**

---

## 2. 我们如何使用你的数据

你的档案数据仅用于：

- 在你访问的求职申请页面上自动识别并填写表单字段
- 通过关键词与语义分析，将表单字段与相应档案条目匹配
- 在启用 AI 时，为开放式申请问题生成草稿答案
- 记住你手动输入的答案，以便下次自动填写同一字段（「自动学习」）

---

## 3. 可选 AI 功能与向第三方传输数据

Apply Pilot 包含一项**可选的 AI 匹配功能**，由第三方大语言模型（LLM）API 提供支持。该功能**默认关闭**，仅在你满足以下条件时才会启用：

1. 在扩展的「设置」页面中明确开启，且  
2. 自行提供来自 Anthropic 或 OpenAI 的 API 密钥。

**在启用 AI 时**，以下数据会发送给你所选 API 服务商（Anthropic 或 OpenAI）：

- 档案中的选定字段（例如职位、工作年限、技能、教育背景、预写答案）
- 当前页面上未匹配表单字段的相关信息（字段标签、占位符、输入类型等）

这些数据由浏览器**直接**发送给 API 服务商，**不会**经由任何 Apply Pilot 服务器中转。

请查阅你所选服务商的隐私政策：

- Anthropic: https://www.anthropic.com/privacy
- OpenAI: https://openai.com/policies/privacy-policy

**若你未启用 AI 功能，则不会有任何数据被发送到外部服务。**

---

## 4. 数据存储与保留

所有数据均通过 `chrome.storage.local` 保存在浏览器本地，并保留在你的设备上，直至你：

- 在「设置」页面中手动清除，或  
- 卸载扩展（将删除所有已存储数据）

Apply Pilot 无法访问你已存储的数据，无法代为读取，也无法代你删除。

---

## 5. 权限说明

| 权限 | 用途 |
|---|---|
| `storage` | 在设备本地保存你的档案与设置 |
| `activeTab` | 读取当前标签页的 URL，以识别正在使用的页面 |
| `scripting` | 在你触发时，将表单填写脚本注入当前页面 |
| `host_permissions: <all_urls>` | 求职申请表单分布在大量雇主与招聘平台网站上。需要较宽泛的主机权限，扩展才能在你选择使用的任意网站上工作。 |

---

## 6. 数据共享

Apply Pilot **不会**：

- 出售、出租或与任何第三方共享你的个人数据  
- 将你的数据传送到任何 Apply Pilot 服务器  
- 将你的数据用于广告或分析  
- 与第三方共享数据，第 3 节所述由你发起的 API 调用除外  

---

## 7. 儿童隐私

Apply Pilot 面向成年求职者。我们不会故意收集未满 16 岁用户的数据。

---

## 8. 本政策的变更

若本政策更新，本页顶部的「最后更新」日期将随之变更。在变更后继续使用本扩展，即表示你接受更新后的政策。
